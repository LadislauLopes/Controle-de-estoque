# README - Programa ArmazenaTi

## Descrição
Este programa foi desenvolvido como parte dos requisitos da disciplina TESI2 na Universidade Federal do Acre (UFAC). O objetivo principal do programa é ser um controle de patrimonio de forma eficiente.

## Execução
Para executar o programa, siga os passos abaixo:

1. Vá para a pasta 'dist' no diretório onde você baixou ou instalou o programa.
2. Abra a pasta ArmazenaTI
3. Localize o arquivo executável chamado 'armazenaTi'.
4. Execute o programa clicando duas vezes no arquivo 'armazenaTi' ou usando o comando apropriado no seu sistema operacional.

## Senha do Gerente
O cadastro ao programa requer uma senha de gerente. A senha do gerente está definida como 'TESI2'.

## Requisitos do Sistema
Certifique-se de que o seu sistema atende aos seguintes requisitos antes de executar o programa:

- Sistema operacional: Windowns
